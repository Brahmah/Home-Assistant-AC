## Pin Connections
* **Wifi LED:** GPIO25
* **AC Reset Pin:** GPIO13
* **AC Cool Button Pin:** GPIO16
* **AC Fan Pin:** GPIO17
* **AC Exhaust Pin:** GPIO18
* **AC Off Pin:** GPIO19
* **AC Fan Up Pin:** GPIO21
* **AC Fan Down Pin:** GPIO22

## Circuit
Pins should go to optocouplers that essentially simulate actual button presses.
